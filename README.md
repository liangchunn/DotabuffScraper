# DotabuffScraper
Scrapes matchup data from dotabuff.com

## Data structure
```
{
    'last-updated' : "dd/mm/yyyy",
    'hero': [
            {'matchup': advantage},
            ...
            ], ...
}
```

## Installation
```
git clone https://github.com/liangchunn/DotabuffScraper
cd DotabuffScraper
npm install
```

## Run
```
npm start
```
